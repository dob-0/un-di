# Projects

Active and past work under di.ii XR studio_network.

## Active

### di.iiii
Spatial authoring platform for immersive XR experiences. The core software project — ongoing development.
- Site: di-studio.xyz
- Repo: github.com/dob-0/di.iiii
- Stack: React + Three.js, Node.js, SQLite, Docker
- Two surfaces: Studio (shipped), Beta (node-graph, in development)
- Long-term direction: decentralized, CRDT op-logs, IPFS-compatible assets, P2P sync

### recordAR
XR art and educational platform connecting artists to explore memory and identity through analog processes and digital storytelling.
- Live: di-studio.xyz/platform_recordAR
- Built with XR Hub Bavaria, Gyumri / Impert-export Munich

### Cascade club
High-fidelity digital recreation of Yerevan's iconic 1990s Cascade Club as an interactive 360° metaverse space.
- Live: yodoor.com/room/2963
- Platform: YODoor Metaverse
